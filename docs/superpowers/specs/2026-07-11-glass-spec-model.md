# 玻璃規格模型與複選欄型 spec(2026-07-11 定案)

延續 2026-07-10-normalization。維護者裁決全數定案,本檔為唯一需求來源。

## 1. 新欄型(schema v4)

`AttributeField.field_type` 擴充:

| 型態 | 說明 | 儲存 |
|---|---|---|
| select | 單選(既有) | 一筆 option_id |
| text | 手打(既有) | 一筆 text_value |
| multi | 複選(勾選框) | 多筆 option_id |
| tags | 特性詞條(逗號分隔輸入,中英逗號皆收) | 多筆 option_id(詞條自動建選項,享改名即生效紅利) |

- `VariantAttribute` 放寬:multi/tags 允許同 (variant_id, field_id) 多筆;select/text 維持單筆(應用層擋+唯一索引)。PK 需重建(migration v4)。
- `AttributeField` 加 `default_option_id`(可 NULL,FK→AttributeOption):建檔時預設帶入。
- 既有 DB 資料不做值轉換——鋼化玻璃直接清掉用新工具重匯(資料源=Excel)。

## 2. 鋼化玻璃的欄配置

| 欄 | 型態 | 選項 | 預設 |
|---|---|---|---|
| 規格 | multi | 亮面/霧面/藍光/防窺 | — |
| 特性詞條 | tags | (自動長) | — |
| 版型(原分類1改名) | select | 滿版/9分滿 | 滿版 |

顯示順位(收銀/商品資料庫/盤點/紀錄/建檔欄位排列):**基礎詞 → 特性詞條 → 版型**,
例 `霧面+藍光｜SGS認證, 無色偏｜滿版`。

## 3. 匯入對照表(鋼化玻璃 24 值,內建於工具)

規格值 → 勾選 + 詞條:

| 原值 | 勾選 | 詞條 |
|---|---|---|
| 亮面/亮面滿版 | 亮面 | — |
| 霧面/霧面滿版 | 霧面 | — |
| 藍光/藍光滿版 | 藍光 | — |
| 防窺/防窺滿版 | 防窺 | — |
| 霧面藍光 | 霧面+藍光 | — |
| 霧面防窺 | 霧面+防窺 | — |
| 抗AR亮面/霧面/藍光/防窺 | 對應勾 | 抗AR |
| 360防窺 | 防窺 | 360度 |
| 霧面防窺(360度) | 霧面+防窺 | 360度 |
| 五倍防窺 | 防窺 | 五倍強化 |
| 電競霧面 | 霧面 | 電競 |
| 亮面藍寶石 | 亮面 | 藍寶石 |
| 9M藍寶石 | 亮面(補預設) | 9M藍寶石(完整一詞,不拆) |
| SGS認證無色偏藍光 | 藍光 | SGS認證, 無色偏 |
| 霧面藍光(SGS認證) | 霧面+藍光 | SGS認證 |
| 低藍光防窺 | 藍光+防窺 | 低藍光 |
| 藍寶石低藍光 | 藍光 | 藍寶石, 低藍光 |

通則:
- 四基礎詞(亮面/霧面/藍光/防窺)皆未出現 → 補勾亮面。
- 低藍光算藍光(勾藍光+詞條低藍光)。
- 版型(分類1):滿版→滿版;**空白→滿版**(新型號未標即滿版);滿版(白)→滿版+詞條`白`;滿版(黑)→滿版+詞條`黑`。
- 詞條寫法統一:去括號、同概念一種寫法(SGS認證/360度/抗AR…)。

## 4. 資料源

改讀 `docs/New產品清單.xlsm`(797 筆、8 種類)。本輪仍只匯 `--category 鋼化玻璃`;
其他種類(插座/行動電源/藍芽耳機/充電線擴充)的欄名與拆法**未定案,不匯**。

## 5. UI

- 建檔頁:multi=勾選框列、tags=文字框(逗號分隔,含該種類既有詞條建議)、select 有預設即帶入。
- 設定頁:欄型維護支援 multi/tags;版型欄可指定預設選項;詞條選項可改名(即全面生效)。
- 錯誤與 guard 慣例照舊;UI 文字正式台灣用語;動 CSS 遞增 `?v=N`。

## 6. 抗AR 備忘

抗AR 僅硬博士出——不設勾選框,留詞條;建議清單會自動帶,不吃虧。
